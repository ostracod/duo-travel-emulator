
= = = DUO TRAVEL EMULATOR = = =

If you find any bugs, please report them to esperantanaso@gmail.com.

To build:

make

To run:

./build/emulator

On Linux you may have to perform:

sudo apt-get install libncurses-dev

Controls:

Arrow keys = Move cursor
Space = Select option
WASD = Scroll through symbols to enter
F = Finish text entry
Escape = Go back
Backspace = Delete
C = Clear text line
Enter = Newline
